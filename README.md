# historia_arduino_alejandro
1. Historia de Arduino
Arduino es una plataforma de electrónica de código abierto (open-source) basada en hardware y software fáciles de usar. Fue creada con el objetivo de facilitar el aprendizaje de la electrónica y la programación para artistas, diseñadores y aficionados.

Origen y fundación
Arduino nació en 2005 en el Instituto de Diseño Interactivo de Ivrea (IDII), en Italia. Fue desarrollado por un equipo liderado por Massimo Banzi, David Cuartielles, Tom Igoe, Gianluca Martino y David Mellis.
El nombre "Arduino" proviene de un bar en Ivrea llamado Bar di Re Arduino, lugar frecuentado por los fundadores del proyecto.
Línea del tiempo
Año	Evento
2005	Creación del primer prototipo de Arduino en el IDII de Ivrea, Italia.
2007	Lanzamiento de Arduino Diecimila, primera versión con conector USB integrado.
2008	Lanzamiento de Arduino Duemilanove con microcontrolador ATmega328.
2010	Lanzamiento de Arduino Mega 2560 y Arduino Nano. La comunidad crece globalmente.
2012	Arduino Due: primera placa con procesador ARM de 32 bits (84 MHz).
2015	Disputa interna entre cofundadores genera Arduino LLC y Arduino SRL. Posterior reconciliación.
2016	Lanzamiento de Arduino 101 con Intel Curie y Arduino MKR1000 con WiFi.
2018	Lanzamiento de Arduino Nano Every y Arduino Nano 33 IoT.
2021-hoy	Arduino expande su ecosistema con placas IoT, IA embebida y entornos cloud.

Impacto y comunidad
Arduino revolucionó la educación tecnológica al democratizar el acceso a la electrónica. Hoy cuenta con millones de usuarios en todo el mundo, una vasta comunidad en línea, y su plataforma es utilizada en colegios, universidades, laboratorios y proyectos profesionales.

2. Periféricos de Arduino y su Uso
Los periféricos son dispositivos externos que se conectan a Arduino para expandir sus capacidades, ya sea para entrada de datos, salida de señales o comunicación.

Periféricos de Entrada
Periférico	Descripción	Uso Común
Sensores de temperatura	LM35, DHT11/DHT22	Monitoreo climático, termostatos
Sensores ultrasónicos	HC-SR04	Medición de distancia, detección de objetos
Sensores de luz (LDR)	Fotorresistencia	Control automático de iluminación
Pulsadores / Botones	Push button	Interacción usuario, control de estados
Sensores de movimiento	PIR HC-SR501	Alarmas, automatización del hogar
Joystick	Módulo KY-023	Control de dirección, videojuegos DIY
Sensor de llama	Módulo infrarrojo	Detección de incendios, seguridad
Sensor de humedad de suelo	Módulo capacitivo/resistivo	Riego automático, agricultura

Periféricos de Salida
Periférico	Descripción	Uso Común
LEDs	Diodos emisores de luz	Indicadores visuales, iluminación
Pantalla LCD 16x2	Display de caracteres con I2C o paralelo	Mostrar mensajes, menús
Pantalla OLED	Display gráfico de alta resolución	Interfaces gráficas compactas
Buzzer / Zumbador	Emisor de sonido activo o pasivo	Alarmas, melodías, notificaciones
Motor DC	Motor de corriente continua	Robótica, vehículos, ventiladores
Servo Motor	Motor con control de posición angular	Brazos robóticos, cámaras PTZ
Motor paso a paso	Stepper motor con driver A4988	Impresoras 3D, CNC, movimiento preciso
Relay	Interruptor electromecánico	Control de cargas de alta potencia (AC)

Módulos de Comunicación
Módulo	Protocolo	Aplicación
HC-05 / HC-06	Bluetooth	Control inalámbrico desde smartphone
ESP8266 / ESP01	WiFi	IoT, conexión a internet, servidores web
NRF24L01	RF 2.4 GHz	Comunicación inalámbrica de largo alcance
SIM800L	GSM/GPRS	Envío de SMS, llamadas, internet 2G
RFID RC522	RFID 13.56 MHz	Control de acceso, identificación
GPS NEO-6M	UART serial	Rastreo de posición geográfica
